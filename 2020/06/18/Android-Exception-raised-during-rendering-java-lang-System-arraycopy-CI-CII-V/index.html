Android Exception raised during rendering: java.lang.System.arraycopy([CI[CII)V

Posted on 2020-06-18 Edited on 2020-06-20 In Android Views: Views: Valine:

问题

在使用EditText时，Graphical Layout 无法预览，并报错Android Exception raised during rendering:java.lang.System.arraycopy([CI[CII)V

原因

SDK API版本为20，这个是Android用于开发可穿戴设备的，不支持EditText。将API改为20之前的就行了。

步骤

第一步：下载低版本的API

若已有低版本的API，可以跳过这一步。
地址：https://www.androiddevtools.cn/

android 4.4W对应的就是API 20的，下载4.4.2版本以下的即可。

第二步：安装低版本的API

若已有低版本的API，可以跳过这一步。
下载并解压后，将解压出的整个文件夹复制或者移动到 your sdk 路径/platforms文件夹，然后打开SDK Manager，打开 Tools(工具)菜单选择 Options(选项)菜单项打开Android SDK Manager Setting对话框，点击 Clear Cache(清除缓存)按钮，然后重启Eclipse(或Android Studio)和SDK Manager。

第三步：更改工程的API版本

右键工程文件夹 ——> Build Path ——> Configure Build Path

在左栏找到Android并单击，在右侧找到Project Build Target，并切换成低版本。

第四步：在Graphical Layout界面切换API版本
